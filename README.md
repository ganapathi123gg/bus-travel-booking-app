# 🚌 Bus Travel Booking App

A full-stack bus travel booking application built with **React** (frontend) and **Django REST Framework** (backend). Users can search buses, view seat availability, book tickets, and manage their bookings.

---

## 🚀 Features

- **User Authentication**: Secure login and registration system
- **Bus Search**: Search buses by route and date
- **Real-time Seat Availability**: View available seats for each bus
- **Online Booking**: Book tickets with seat selection
- **Booking Management**: View and manage personal bookings
- **Admin Dashboard**: Admin panel for managing buses and bookings
- **Responsive UI**: Mobile-friendly interface

---

## 🛠 Technology Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | React, Vite, Redux, CSS |
| **Backend** | Django, Django REST Framework |
| **Database** | SQLite (can be upgraded to PostgreSQL) |
| **API** | REST API |
| **Authentication** | JWT / Session-based |
| **Version Control** | Git & GitHub |
